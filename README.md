# SimpleWindow v0.0.1
- C++で作成したWindows専用ライブラリです
- シンプルか？と言われるとそうでもない気もしますが、ご了承ください
- (AIを使用して修正したコードが含まれています)

# Requirement
- VC++14

# 使い方
- 基本的に SimpleWindow.h をインクルードすれば直ぐ使えます
- 静的ライブラリを生成して使うこともできます
- main.cppに使用例を載せています
